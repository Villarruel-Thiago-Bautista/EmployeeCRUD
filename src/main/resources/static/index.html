Person CRUD

What is it?

It's a Java-based REST API CRUD using Spring Boot as the framework, where I applied design patterns such as: MVC, DTO, Repository, IoC, and Dependency Injection. I also implemented data validation and custom exception handling.
The frontend is built with HTML, CSS, and JavaScript.
It uses a relational SQL database, I used MySQL as the database management system, and is version-controlled with Git.
The endpoints were tested with Postman and are documented using Swagger.

Create

Fill in the fields with the corresponding person's information to load it into the database, and they will be displayed in a table below.

ID Name Lastname Email Gender DNI Age Cellphone

Update

Specify the ID corresponding to the person you wish to modify. Fill in the fields with the new information to update them in the database and in the table.
(DO NOT LEAVE BLANK SPACES; IN CASE, FOR EXAMPLE, THE NAME REMAINS THE SAME, ENTER IT AS IT IS)


Read

You can list the data of a person by their ID, of all persons by their gender, or simply of all persons loaded in the database.

Delete

Specify the ID corresponding to the person you want to delete from the database, and at the same time, the table on the screen will be updated.
