CRUD de personas

¿Qué es?

CRUD el cual permite ingresar datos de personas a una base de datos, así como también consultarlos, eliminarlos y/o modificarlos.
El backend está hecho en Java utilizando Spring boot como framework, apliqué los patrones de diseño MVC, DTO, inyeccion de dependencias, Repository, e IoC (Inversion of Control).
El frontend está hecho con HTML, CSS y JavaScript, el tipo de base de datos es relacional y el gestor de base de datos utilizado es MySQL.
Los endpoints fueron testeados con Postman.

Registrar

Rellene los campos con la información correspondiente de la persona para cargarla en la base de datos, además, estos se mostrarán a continuación en un tabla.

ID Nombre/s Apellido/s Email Genero DNI Edad Celular

Modificar

Indique el ID correspondiente a la persona que desea modificar. Rellene los campos con la nueva informacion para actualizarlos en la base de datos y en la tabla.
(NO DEJE ESPACIOS EN BLANCO, EN CASO DE QUE POR EJEMPLO EL NOMBRE PERMANEZCA IGUAL, VUELVA A COLOCARLO TAL CUAL)


Listar

Puede listar los datos de una persona por su ID, de todas las personas por su género, o simplemente de todas las personas cargadas en la base de datos

Eliminar

Indique el ID correspondiente a la persona que desea eliminar de la base de datos, a su vez, se actualizará la tabla en pantalla.

Si desea eliminar los datos de todas las personas, clickee en el siguiente botón
ESTE BOTÓN ELIMINARÁ LOS DATOS DE TODAS LAS PERSONAS
