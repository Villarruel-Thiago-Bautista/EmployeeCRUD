Gestor de empleados

Pequeño CRUD el cual permite ingresar empleados a una base de datos, así como también listarlos, eliminarlos y/o modificar sus datos.
El backend está hecho en Java utilizando Spring boot como framework, el frontend está hecho con HTML, CSS y JavaScript, el tipo de base de datos es relacional y el gestor de base de datos utilizado es MySQL.
Apliqué los patrones de diseño MVC y DTO.

Registrar los datos de un nuevo empleado

Rellene los campos con la información correspondientes para cargarlos en la base de datos, además, se actualizarán a continuación en un tabla.

ID Nombre/s Apellido/s Email Género DNI

Modificar los datos de un empleado

Indique el ID correspondiente al empleado que desea modificar. Rellene los campos correspondientes con la información actualizada para actualizarlos en la base de datos y en la tabla.

Listar datos

Puede listar los datos de un empleado por su ID, de todos los empleados por su género, o simplemente de todos los empleados cargados en la base de datos

Eliminar datos

Indique el ID correspondiente al empleado que desea eliminar.

Si desea eliminar los datos de todos los empleados, clickee en el siguiente botón
